# SongBuilder Pro

Professional browser-based music producer built with Next.js, React, Tone.js, and the Web Audio API.

## Features

- Multi-track arrangement timeline with clips, loop regions, and playhead
- Drum machine step sequencer (16 steps)
- Piano roll MIDI editor
- Per-track mixer (volume, pan, mute, solo) and master bus
- Live playback via Tone.js transport
- WAV and MP3 export
- Demo session included on launch

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000/studio](http://localhost:3000/studio).

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Tone.js
- lamejs (MP3 export)
